HIGHH-6 · 접근성

⏸ 의도적으로 변경하지 않음 (거짓양성 · 스코프크리프)

"모두 수정"은 거짓양성을 가려내 "검토함·변경없음"으로 보고하는 것을 포함합니다.

  • 토큰 명명 차이 (--blue-500 ↔ --moisho-blue-500, --shadow-glow ↔ --glow-blue) — index.html은 자체 변수를 쓰는 독립 페이지. 값은 일치 → 시스템 깨짐 아님, 변경 시 무위험 이득.
  • index.html clamp() 반응형 — 고정 px 토큰 문서와 다른 건 결함이 아님.
  • verified 엔드포인트 일괄 403 추가 — OpenAPI 장황화, 가치 낮음.
  • 고아 스키마 삭제 (Attendance/PaymentReady/PaymentResult/Refund) — 계약 파일에서 코드 삭제는 승인 사안 → 권고만(옵션).
  • docs/01 PRD 서사 · docs/04 상세 전면 재작성 — 거대·고위험, README caveat + 배너가 커버 → 옵션으로 남김.
  • /admin/deposits · /admin/payouts/decision — 에스크로와 긴장 있으나 운영 분쟁/대사용일 수 있어 사용자 판단으로 보류.
