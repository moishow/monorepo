AP-01

로그인 · 가입

온보딩

신규 사용자가 소셜 계정으로 3초 만에 진입하고, 본인인증(KYC)을 거쳐 가상 지갑을 개설하는 출발점.

구성 요소
  1. 1브랜드 로고/태그라인 — 서비스 정체성을 노출하는 진입 화면.
  2. 2카카오 로그인 (주 CTA) — 탭 시 카카오 OAuth → 최초 1회 본인인증(KYC) 진입.
  3. 3이메일 가입/로그인 — 소셜을 쓰지 않는 사용자용 보조 경로.
사용자 동작 · 인터랙션
  • 카카오/애플 탭 → 인증 → (신규)KYC → 홈 / (기존)홈
  • 이메일 탭 → 이메일·비밀번호 입력 폼

발견

푸시 발송

신규

타겟을 골라 모이쇼 보이스로 푸시 알림을 보내고, 실시간 미리보기·발송 이력을 관리.

구성 요소
  1. 1타겟 선택 — 전체·미입금자·총무·휴면 등 세그먼트.
  2. 2메시지 작성 — 보이스 프리셋 + {이름} 변수 치환.
  3. 3실시간 폰 미리보기 — 실제 알림 표시를 즉시 확인.
  4. 4발송/예약 — 즉시 발송 또는 예약. 이력·열람률 기록.

Showts 검수

신규

숏폼 영상 전용 검수 큐. 쇼 피드와 분리해 승인된 영상만 노출.
